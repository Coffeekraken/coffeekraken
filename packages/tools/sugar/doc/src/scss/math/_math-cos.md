# math-cos

<!-- @namespace: sugar.scss.math.math-cos -->

Type : **{ function }**


Calculate the cosinus of the passed angle



### Parameters
Name  |  Type  |  Description  |  Status  |  Default
------------  |  ------------  |  ------------  |  ------------  |  ------------
$angle  |  **{ Number }**  |  The angle to calculate the cosinus from  |  required  |

Return **{ Number }** The calculated cosinus

### Example
```scss
	sugar.math-cos(10); // => 6.6645643378
```

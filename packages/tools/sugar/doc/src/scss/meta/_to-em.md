# to-em

<!-- @namespace: sugar.scss.meta.to-em -->

Type : **{ function }**


Convert the passed value to em



### Parameters
Name  |  Type  |  Description  |  Status  |  Default
------------  |  ------------  |  ------------  |  ------------  |  ------------
$value  |  **{ Number }**  |  The value to convert  |  required  |

Return **{ Number }** The em converted value

### Example
```scss
	sugar.to-em(10px);
```

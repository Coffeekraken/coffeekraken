# vertical-rhythm

<!-- @namespace: sugar.scss.core.mixin.vertical-rhythm -->

Type : **{ mixin }**


Scope the content inside the `settings.vertical-rhythm.scope-class` class


### Example
```scss
	margin-bottom: 2em; // will be applied only if .my-cool-item is in the .vr scope
	}
}
```

# math-tan

<!-- @namespace: sugar.scss.math.math-tan -->

Type : **{ function }**


Calculate the tangent of the passed angle



### Parameters
Name  |  Type  |  Description  |  Status  |  Default
------------  |  ------------  |  ------------  |  ------------  |  ------------
$angle  |  **{ Number }**  |  The angle to calculate the tangent from  |  required  |

Return **{ Number }** The calculated tangent

### Example
```scss
	sugar.math-tan(10); // => 0.4142942863
```

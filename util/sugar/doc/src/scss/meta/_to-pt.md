# to-pt

<!-- @namespace: sugar.scss.meta.to-pt -->

Type : **{ function }**


Convert the passed value to pt



### Parameters
Name  |  Type  |  Description  |  Status  |  Default
------------  |  ------------  |  ------------  |  ------------  |  ------------
$value  |  **{ Number }**  |  The value to convert  |  required  |

Return **{ Number }** The pt converted value

### Example
```scss
	sugar.to-pt(10px);
```

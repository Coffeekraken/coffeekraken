# to-px

<!-- @namespace: sugar.scss.meta.to-px -->

Type : **{ function }**


Convert the passed value to px



### Parameters
Name  |  Type  |  Description  |  Status  |  Default
------------  |  ------------  |  ------------  |  ------------  |  ------------
$value  |  **{ Number }**  |  The value to convert  |  required  |

Return **{ Number }** The px converted value

### Example
```scss
	sugar.to-px(10rem);
```

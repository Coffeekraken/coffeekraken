# to-rem

<!-- @namespace: sugar.scss.meta.to-rem -->

Type : **{ function }**


Convert the passed value to rem



### Parameters
Name  |  Type  |  Description  |  Status  |  Default
------------  |  ------------  |  ------------  |  ------------  |  ------------
$value  |  **{ Number }**  |  The value to convert  |  required  |

Return **{ Number }** The rem converted value

### Example
```scss
	sugar.to-rem(10px);
```

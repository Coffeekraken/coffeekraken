# vertical-aligner

<!-- @namespace: sugar.scss.mixin.vertical-aligner -->

Type : **{ mixin }**


Vertically align content by using the inline-block middle trick


### Example
```scss
	.my-cool-container {
	@include sugar.vertical-aligner();
}
```

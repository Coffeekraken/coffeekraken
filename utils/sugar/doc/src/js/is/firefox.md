### Example
```js
	import isFirefox from '@coffeekraken/sugar/js/is/firefox'

}
```
Return **{ [Boolean](https://developer.mozilla.org/fr/docs/Web/JavaScript/Reference/Objets_globaux/Boolean) }** true if is firefox, false if not

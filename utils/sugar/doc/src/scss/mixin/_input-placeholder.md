# input-placeholder

<!-- @namespace: sugar.scss.mixin.input-placeholder -->

Type : **{ mixin }**


Set the placeholder style for inputs, etc...


### Example
```scss
	input, textarea {
	@include sugar.input-placeholder() {
		color : red;
	}
}
```

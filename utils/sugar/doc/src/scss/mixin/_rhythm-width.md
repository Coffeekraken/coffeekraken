# rhythm-width

<!-- @namespace: sugar.scss.mixin.rhythm-width -->

Type : **{ mixin }**


Print out the css on a container that will make the lines displays a maximum number of characters.


### Example
```scss
	h1 {
	@include sugar.rhythm-width(45);
}
```
